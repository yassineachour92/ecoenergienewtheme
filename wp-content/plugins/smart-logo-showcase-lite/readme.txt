<strong>Smart Logo Showcase Lite</strong> - is a WordPress logo showcase plugin packaged with 5 beautiful pre-designed templates. With this fully responsive logo showcase builder, you can display logo images, however you like and configure it to the most. Use different layouts available and showcase your client/ sponsor/ partner/ brand logo in a visually appealing way.

<strong>Smart Logo Showcase Lite</strong>  - is developed considering user friendliness, features, layout designs for various users.

== Installation ==
1. Unzip smart-logo-showcase-lite.zip
2. Upload all the files to the /wp-content/plugins/smart-logo-showcase-lite.
3. Activate the plugin through the 'Plugins' menu in WordPress.
4. For customizing the plugin's settings, click on Smart Logo option menu in Wordpress left admin menu.

== Frequently Asked Questions ==
= What does this plugin do? =
This plugin provides the ability to showcase your client/ sponsor/ partner/ brand logo in a visually appealing way.

= Can I have multiple logo showcase with seperate setting in single page or post?  =
Yes, you can have unlimited logo showcase with each seperate setting in single page , post or template.

= How many logo showcase can be created with a plugin? =
You can create logo showcase as many as you want.

= Is this plugin compactible with all themes?  =
Yes, this plugin is compactible with all themes which have followed the wordpress theme development standards.

= How do I use the smart logo showcase Lite? =
You just need to create a smart logo showcase lite from backend and the generated shortcode can be used in your posts , pages or templates.
